# WeSee — Product Overview

WeSee is an AI content platform that helps teams generate, review, and publish
marketing content. The core product has three surfaces: the Studio (where users
draft and edit content), the Review Queue (where teammates approve drafts), and
the Publish API (which pushes approved content to connected channels).

WeSee supports three content types: short-form social posts, long-form articles,
and image captions. Every generation is grounded in a user-provided brief and the
team's saved brand voice profile. Generated drafts always start in the "Draft"
state and must be moved to "Approved" before they can be published.

WeSee does not train its base models on customer content. Customer content is used
only to build per-team retrieval indexes, which are isolated per workspace.
